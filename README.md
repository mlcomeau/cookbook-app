# cookbook-app

An app where chefs can store and share recipes. 

## Installation

`````
git clone https://github.com/mlcomeau/cookbook-app
cd cookbook-app/backend/cookbook 
rails db:migrate 
rails db:seed 
rails s 
`````

## Endpoints

http://localhost:3000/chefs

http://localhost:3000/recipes
